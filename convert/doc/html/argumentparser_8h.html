◆ argToChar()

void argToChar ( char *  str,
char *  output 
)

Stores the identifier character for the given string in the given pointer.

The identifier character is the second character of the argument, for the arguments of the form "-x", where x is any character.

Parameters
strA string, the argument to check. Valid arguments are "-n", "-s", "-d", "-h", and "-v".
outputA pointer to char, where the output will be stored. If the input argument is not valid, the output is a null character.

void validateDigit ( const char *  digit,
const int *  base,

Checks if the given character is a valid digit for a given base.

Parameters
digitA pointer to char, the character to check.
baseA pointer to integer, pointing to a value in the range [2, 16], the base to check.
controlA pointer to integer. This procedure will store EXIT_FAILURE on it if an error occurred.
Attention
Calling this procedure with base outside range [2, 16] is undefined behavior.
